# Gesunrin Cafe

Static front-end cafe ordering prototype.

## Open
Open `index.html` in a browser.

## Flow
Catalog -> Cart -> Checkout -> Payment -> Invoice -> Order History

## Admin Login
Username: `admin`
Password: `admin123`

## Pages
- `index.html` - home
- `catalog.html` - catalog
- `product-detail.html` - product detail
- `cart.html` - cart
- `checkout.html` - customer details
- `payment.html` - payment method
- `invoice.html` - invoice and queue number
- `history.html` - order history
- `articles.html` - article archive
- `article-detail.html` - article detail
- `login-admin.html` - admin login
- `admin.html` - dashboard
- `admin-articles.html` - manage articles
- `admin-products.html` - manage products
- `admin-users.html` - manage users
- `admin-transactions.html` - manage transactions
